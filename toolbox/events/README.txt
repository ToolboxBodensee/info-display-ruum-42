es ist in diesem node moeglich, bis zu 7 zeilen text auf jedem bild anzuzeigen.
hierzu einfach eine datei anlegen, die exakt so heisst wie das bild, plus ".url"
hinten dran. in dieser muss eine url liegen, deren aufruf bis zu 7 zeilen text
zurueckliefert.

beispiel: euer-bild.png         <-- das bild
          euer-bild.png.url     <-- da drin steht die url die den text liefert

die 7 zeilen werden zentriert auf das hintergrundbild geschrieben. wollt ihr
also z.b. nur 1 zeile text in der mitte des bildschirms haben, dann sollte der
text einfach in der 4. zeile stehen und die anderen zeilen sollten leer sein.
